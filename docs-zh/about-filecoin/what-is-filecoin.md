# Filecoin 是什么?

Filecoin 是一个拥有内置贡献奖励机制的去中心化网络，可确保文件长期可靠地存储。
在 Filecoin, 用户需要为miner存储设备付费. miner设备是负责储存文件及校验文件长期可用的计算机设备。 任何想要存储文件或者提供他人有偿文件储存服务的用户都可以加入 Folecoin。可用的存储空间及存储空间的价格不受任何公司或组织的控制。取而代之的是 Filecoin 为所有需要存储或检索使用文件的用户提供了便利。
Filecoin 的区块链记录了要发送和接收 FIL 的交易，以及miner的证明，证明他们正确地存储了文件。

## 用户

Filecoin 允许用户以极具竞争力的价格存储他们的文件，并验证他们的文件的准确性。

用户可以通过选择存储空间最适合自己需要的miner，在成本、冗余和速度之间进行权衡。实现 Filecoin 的应用程序可以与网络上的任何miner存储兼容。与集中式存储系统不同，不需要为每个提供者实现不同的 API。
用户任何时候都可以通过查看 Filecoin 的块链证明来验证他们的文件是否正确地存储。

## 存储供应商

Filecoin 允许存储供应商在市场上公开出售他们的存储服务。

存储提供商可以是经营miner的个人或组织，他们通过这样做赚取 Filecoin 代币。一个miner可以是任何连接到互联网的，且带有备用磁盘空间的计算机，或是一个专门为 Filecoin 建立的具有大量存储空间的专用系统。 Filecoin 的区块链奖励为互联网提供有效存储的miner，反之对完成无用的计算证明的miner则没有任何奖励。

一旦存储提供者接受了 Filecoin 的协议，他们就可以访问整个 Filecoin 用户市场。提供商不需要设计他们自己的存储 API 或者宣传他们自己的产品，因为这是由 Filecoin 的协议和网络处理的。通过降低进入市场的门槛，Filecoin 驱使独立存储提供商们组成一个活跃的、分布式的生态系统。

如果你刚接触 Filecion 或 web 3, 我们非常欢迎您的到来! 我们认为您会喜欢了解使 [IPFS](https://ipfs.io), [Filecoin](https://filecoin.io), [Powergate](https://github.com/textileio/powergate), 和其他在你的项目和应用中可能会使用到的重要技术，以及应该如何使用它们。

Web3 是一个运作在软件开发过程中将集中式应用程序转变为分散式协议的驱动者。建立在 web3 架构上的应用程序不必具有单一控制点，通过消除中间角色降低大公司审查的风险，并降低了服务遭受拒绝服务攻击的风险。Web 3 是一个不受信的基础设施，允许用户掌握所有权并收回对其数据的控制权。
Web3 本质上是通过允许在同级之间共享文件来运作 Filecoin 的。

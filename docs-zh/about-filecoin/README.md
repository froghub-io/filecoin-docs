# 引言

Filecoin 的使命是为人类的信息创造一个分散的、高效的、健壮的基础。为了推进这项任务，我们创建了一个分散式存储网络，让世界上的任何人都可以存储或检索文件。我们希望 Filecoin 能够帮助小型企业直接与知名的集中服务竞争，为各种规模的供应商提供通用接口，并大大降低迁移和交易成本，从而降低存储成本，提高存储的可靠性。

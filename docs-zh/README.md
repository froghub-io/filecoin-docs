# Filecoin 文档

**Filecoin 是一个 P2P 网络，可将文件存储在 Internet 上，并具有内置的经济激励机制，可确保随着时间的推移可靠地存储文件。**

可用的存储空间和价格不受任何一家公司的控制。相反，Filecoin 推进了存储文件和检索文件的市场的开放，使得任何人都可以参与进来。
